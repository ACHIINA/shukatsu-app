# 就活補助アプリ 要件定義

## プロダクト概要

就活生の「だるい作業」を削減し、本質的な準備に集中させる就活補助ツール。
Mac/Windowsのデスクトップアプリをメインに、iOS補助アプリを提供する。

**ターゲットユーザー:** 日本の大学生・大学院生（新卒就活中）
**主要配布チャネル:** 大学への直接営業（中央大学→MARCH・早慶へ展開）
**価格:** 無料 or 100円（競合は600円・100DL程度で市場未開拓）

---

## 機能一覧

### メインアプリ（Mac / Windows デスクトップ）

#### 1. スケジュール管理
- 選考ステータスの一元管理（エントリー・ES提出・説明会・面接・結果）
- Googleカレンダー連携（面接日程の自動登録）
- 締め切りリマインダー通知

#### 2. メール確認・整理
- Gmail API連携
- 企業からのメール自動分類（説明会案内・選考通知・合否等）
- 重要メールのハイライト表示

#### 3. 企業調査
- 企業名入力 → AI自動調査（事業内容・業績・求める人物像・競合）
- 志望動機のヒント生成
- 業界比較

#### 4. ES（エントリーシート）下書き補助
- 設問入力 → AI下書き生成
- 自己分析データ（一度入力で使い回し）
- 過去ESの管理・再利用

### 補助アプリ（iOS）

- スケジュール確認（デスクトップと同期）
- 面接リマインダープッシュ通知
- 簡易メモ（面接後の振り返りメモ等）

---

## 対象外（スコープ外）

- 就活サイト（マイナビ・リクナビ等）への自動応募 → 対象外
- 企業個別サイトへの自動登録 → 対象外（手動補助にとどめる）

## 技術スタック（案）

| 領域 | 技術候補 |
|------|---------|
| デスクトップ（Mac/Win） | Electron + React または Tauri + React |
| iOS補助アプリ | Swift / SwiftUI |
| AI機能 | Claude API または OpenAI API |
| メール連携 | Gmail API（OAuth2） |
| カレンダー連携 | Google Calendar API |
| データ同期（デスクトップ↔iOS） | Firebase または Supabase |
| 認証 | Google OAuth |
